# DesktopManagerStock

Sistema de gestion de invesntario y stock para escritorio,
desarrollado con [Python/Java] + [APIS]

# Caracteristimas

- Gestion de productos (alta, baja, modificacion)
- Control de stock mínimo y máximo
- Entradas y salidas de inventario
- Reportes en PDF/Excel
- Alertas de stock bajo
- Cálculo de ganancias/pérdidas
- Impresion de etiquetas y comprobantes
- Interfaz linda
- Autentificacion segura de usuario

# Tecnologias utilizadas

### Backend (Python)
- Python 3.11+
- FastAPI
- SQLAlchemy
- SQLite

### Frontend (Java)
- Java 17+
- Swing
- HttpClient (API Consumible)

### Herramientas
- Git & Github
- Maven
- Postman (tests)

## Requisitos previos

- Python 3.11+
- Java JDK 17+
- Maven 3.8+
- SQLite3
- Git (opcional)
